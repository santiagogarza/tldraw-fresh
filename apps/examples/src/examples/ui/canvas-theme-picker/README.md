Canvas theme picker.

---

This example adds a five-mode canvas theme picker to the top-left menu bar. Click
the circle next to the main menu to switch between light, dark, sky, sunrise,
and sunset. The active mode updates the canvas background and accent colors, and
persists across page reloads.
